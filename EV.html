VIDA VCI

Diagnostic Tool for Electric 2 wheeler ECU’s

Core Features

  • • PC / Laptop interfaced Flashing Tool
  • • Online/Offline flashing mode
  • • Multiple Protocols & Automatic Protocol selection
  • • Inbuilt Power supply, no external charging required
  • • Complete diagnostic solutions enabled
  • • DTC’s with Cause & Remedial action
  • • Adjust VIN number.
  • • XML Flashing for configuring ECU
  • • 2D / 3D Scanner Configuration supported
  • • Database Management System (DBMS) enabled
  • • Online firmware & Software updation
  • • VCI connect to Laptop / PC via USB / Bluetooth
  • • Telematics Configuration

Software Features

  • • Software Selectable options for ECU’s
  • • Online update of Firmware
  • • Automatic Protocol (CAN / KWP) Selection
  • • Vehicle Diagnostics – Actual Parameter, DTC’s, Adjust parameters, Test Actuators, Freeze frame, Data select, Graph Mode.
  • • Online and Offline ECU Flashing Support
  • • Dataset download from Cloud.
  • • VIN Writing & Reading option enabled
  • • 2D / 3D Scanner integration
  • • Multiple ECU variants can be merged in the same VCI
  • • Runtime Data logging
  • • Data select Option for Individual parameter selection
  • • Graph Mode for Graphical Analysis.

Specification

  • • Size W x H x D - 90 x 130 x 40 mm.
  • • Weight - 280 Gram (Approx)
  • • Body Color – Orange, Black & White
  • • PC Sync - USB Port (3.0)
  • • Version – 3.0
  • • Language - English
  • • Product Type - Compact
  • • User Manual

Hardware features

  • • Laptop / PC based Diagnostics Tool
  • • External Communication via K – Line / CAN
  • • CAN Communication via J1939 Protocol / UDS
  • • Power supply range 08 Volt – 32 Volt
  • • 32 Bit, 120 MHz ARM Cortex Series Processer
  • • In built 09 volt Power Supply
  • • J1962M to DB9F, OBD II Cable
  • • Uploading the data to PC via USB 3.0
  • • EMI/ EMC protection enabled
